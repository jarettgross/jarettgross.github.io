Description

I implemented the following using Java: a divide and conquer
closest pair of points algorithm running in time O(nlogn);
a hash table that maps strings to specific positions in the
table based on the hashing; Dijkstra's shortest paths algorithm
that computes the shortest paths in a graph; a skip list which
performs insertion, deletion, and find in runtime O(logn);
A logistic regression algorithm that utilizes gradient descent
to learn a model on data.
