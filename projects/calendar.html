Description

Together with Brennan Morell, we created a calendar which can
keep track of users and allow each each user to have their own
calendar full of events. These events can be tagged with a
category, shared with other users, edited, and deleted.
JavaScript was used for the front-end of the site, while
PHP was used as the back-end. mySQL was used to store all
of the events and users.
